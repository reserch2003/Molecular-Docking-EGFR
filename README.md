# Molecular-Docking-EGFR

##Objective
To investigate the binding interaction of Gefiitinib with the EGFR ATP-binding pocket using molecular docking.

##Software Used
-AutoDock/FitDock
-PyMOL

##Result
-Binding affinity: -7.3 kcal/mol
-Hydrigen bond interaction observed
-Aromatic interaction contribute to stabilization
